# Web Projects – GitHub Spark Portfolio

**Kompakte Web-Anwendungen zur praktischen Übung moderner Frontend-Architekturen und UI-Designs**

Dieses Repository fasst mehrere **kleine, produktionsnahe Web-Projekte** zusammen, die im Rahmen meines persönlichen Trainings zur Entwicklung moderner Web-Anwendungen entstanden sind.

Der Fokus liegt auf:

* sauberer Frontend-Architektur
* konsistenter UI-/UX-Gestaltung
* moderner React-Toolchain
* praxisnaher Umsetzung (State, Persistenz, Internationalisierung)

Alle Projekte sind bewusst **überschaubar gehalten**, um einzelne Konzepte gezielt zu trainieren.

## 📝 WebInsait Text Editor

**Webbasierter Rich-Text-Editor mit Notizbuch-Optik**

WebInsait Text Editor ist ein eleganter, browserbasierter Texteditor zum Schreiben, Strukturieren und Exportieren von Texten. Das Projekt stellt die **Web-Version eines bestehenden Desktop-Projekts** dar und dient als Vergleich zwischen Desktop- und Web-Architekturen.

### Kernfunktionen

* Rich-Text-Formatierung (Fett, Kursiv, Unterstrichen, Farben)
* Dokumentenverwaltung (Erstellen, Bearbeiten, Löschen, Suchen)
* Export als PDF und TXT
* Linierter und karierter Papierhintergrund
* Responsive Design für Desktop und Mobile

### Technologie-Stack

* React 19 + TypeScript
* Vite
* Tailwind CSS + shadcn/ui
* Spark KV Storage

🔗 **Web-App:** [https://webinsaittexteditor--Developer3421.github.app](https://webinsaittexteditor--Developer3421.github.app)

## ♟️ Chess & Translate

**Schachspiel mit integrierter Mehrsprachen-Kommunikation**

Chess & Translate kombiniert ein vollständiges Schachspiel mit einem integrierten Übersetzungssystem. Ziel ist es, Spielern aus unterschiedlichen Sprachräumen eine gemeinsame Spiel- und Kommunikationsbasis zu bieten.

### Kernfunktionen

* Vollständige Schachregeln inkl. Rochade, En-passant und Bauernumwandlung
* Mehrsprachige Benutzeroberfläche
* Echtzeit-Übersetzung von Benutzertexten
* Persistente Speicherung von Spielständen und Statistiken
* Responsive Layout für Desktop und Mobile

### Technologie-Stack

* React 19 + TypeScript
* Vite
* Tailwind CSS + shadcn/ui
* Spark Runtime SDK (LLM + KV Storage)

🔗 **Web-App:** [https://chess-translator-app--Developer3421.github.app](https://chess-translator-app--Developer3421.github.app)

## 🔑 Password Generator

**Sicherer und anpassbarer Passwort-Generator im Browser**

Password Generator ist eine schlanke Web-Anwendung zur Erstellung starker, zufälliger Passwörter direkt im Browser. Der Nutzer kann Länge, Zeichensatz (Großbuchstaben, Kleinbuchstaben, Ziffern, Sonderzeichen) und weitere Parameter individuell konfigurieren und das generierte Passwort mit einem Klick kopieren.

### Kernfunktionen

* Einstellbare Passwortlänge und Zeichenkategorien
* Sofortige Stärke-Bewertung des generierten Passworts
* Ein-Klick-Kopieren in die Zwischenablage
* Vollständig clientseitig – keine Daten verlassen den Browser
* Responsive Design für Desktop und Mobile

### Technologie-Stack

* React 19 + TypeScript
* Vite
* Tailwind CSS + shadcn/ui

🔗 **Web-App:** [https://password-generator--Developer3421.github.app](https://password-generator--Developer3421.github.app)

## Ziel dieses Repositories

Dieses Repository dient als **praktisches Übungsfeld für produktionsnahe Web-Entwicklung**.

Trainiert werden insbesondere:

* Aufbau skalierbarer React-Anwendungen

* Internationalisierung und Übersetzungskonzepte
* UI-Konsistenz und Design-Systeme
* Vergleich zwischen Desktop- und Web-Anwendungen
